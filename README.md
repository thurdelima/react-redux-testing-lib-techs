## Run Locally

Clone the project

```bash
  git clone https://github.com/thurdelima/react-redux-testing-lib-techs
```

Go to the project directory

```bash
  cd react-redux-testing-lib-techs
```

Install dependencies

```bash
  yarn install or yarn
```

Start the server

```bash
  yarn start
```  

  Run tests

```bash
  yarn test
```
